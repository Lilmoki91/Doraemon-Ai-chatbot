# Doraemon AI Chatbot

Doraemon AI Chatbot is an open-source web-based chatbot inspired by the popular anime character Doraemon. This project leverages **HTML**, **CSS**, and **JavaScript** to create an engaging and interactive experience for users.

## Features

- **Web-Based Application**: Accessible through any web browser.
- **Interactive Design**: Engaging conversations powered by JavaScript.
- **Customizable UI**: Easily editable HTML and CSS files for a personalized look and feel.
- **Open Source**: Contributions are welcome to improve the project.

## Getting Started

Follow these instructions to get a copy of the project up and running locally.

### Prerequisites

To run this project, you need:

- A modern web browser (e.g., Chrome, Firefox, Edge).

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Lilmoki91/Doraemon-Ai-chatbot.git
   cd Doraemon-Ai-chatbot
   ```
2. Open the `index.html` file in your browser to launch the chatbot.

## Usage

- Start the chatbot by opening the index file in your browser.
- Chat with Doraemon by typing your messages into the input provided.
- The chatbot's responses are generated in real-time.
